A methodology of steel microstructure recognition using SEM images by machine learning based on textural analysis

宇都恭平
October 2020
URL

Abstract

熱処理の異なる8種の鉄鋼組織のSEM(Scanning Electron Microscope)画像に対してGLCM(Gray-Level Co-occurrence Matrix)というテクスチャ特徴量を用いてGBDTで組織分類を行なった研究です。修士2年の8月から実験を開始し、修論として執筆しました。卒業後も引き続き論文の執筆に携わり2020年10月にジャーナルに投稿しました。(3rd Author)
